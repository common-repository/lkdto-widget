== Description ==

Easily display your [lkd.to](http://lkd.to) links in your WordPress widget areas.

== Installation ==

1. Upload unzipped plugin directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Activate the widget through the 'Appearance - Widgets' menu in WordPress

== Frequently Asked Questions ==

= What's lkd.to? =
More information can be found [here](http://lkd.to).

= I changed something in my lkd.to profile. Why don't I don't see the changes in the widget? =
The widget will update from the lkd.to site every three hours. You can also manually make the widget refresh by clicking the "Save" button in the widget's administration panel.
